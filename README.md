# Ai-Coder-Journey

A personal repository documenting the path to becoming an AI-assisted developer — learning how to work effectively with coding agents, build real projects incrementally, and ship production-ready software.

## Current Project

**Kanban MVP** — a full-stack Kanban board application with boards, columns, tasks, drag-and-drop, authentication, and persistence.

AI coding agents working in this repo should read [AGENTS.md](AGENTS.md) before making changes. That file contains the MVP roadmap, architecture guidelines, and implementation standards.

## Goals

- Learn to collaborate with AI coding agents effectively
- Build projects incrementally with clean architecture and reusable components
- Document decisions, patterns, and lessons along the way
- Ship a working MVP before adding advanced features
- Develop production-ready habits: testing, security, and maintainability

## Roadmap

1. **Setup** — Repository structure, agent instructions, and tooling
2. **Kanban MVP** — Foundation, auth, boards, columns, tasks, drag-and-drop
3. **Patterns** — Reusable components, state management, API design
4. **Production habits** — Testing, error handling, deployment, documentation

## For AI Agents

Read [AGENTS.md](AGENTS.md) before writing any code. It defines the MVP scope, phased roadmap, tech stack, and quality standards for this project.

## Getting Started

This repo is in early setup. Implementation begins with Phase 1 (Project Foundation) as outlined in AGENTS.md. More code and documentation will be added as the Kanban MVP takes shape.
